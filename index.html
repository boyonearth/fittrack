Connect to GitHub to sync your data across all devices. Your data saves to fittrack-data.json in your repo automatically.

  1. Go to github.com → Settings → Developer settings
  2. Click Personal access tokens → Tokens (classic)
  3. Click Generate new token (classic)
  4. Give it a name: FitTrack
  5. Tick repo checkbox (full repo access)
  6. Click Generate token
  7. Copy the token and paste below
